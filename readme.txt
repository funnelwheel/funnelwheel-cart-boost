Cart Boost for WooCommerce delivers mobile-first, CRO-optimized site experiences that help you sell more and increase customer retention.

== Description ==

**FunnelWheel Cart Boost** is a performance-focused extension for WooCommerce that enhances your cart with optimized funnels, rewards, and upsells — improving conversions and customer experience.

Key Features:

* Smart mini cart with upsell & cross-sell
* Gamified reward progress bar
* Conversion-optimized cart layout
* Dynamic product suggestions
* Funnel & flywheel integration (via FunnelWheel platform)

Built to be lean, fast, and mobile-first — FunnelWheel Cart Boost helps merchants increase average order value and customer lifetime value without extra bloat.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` or install via WordPress admin.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Go to **WooCommerce → Cart Boost** settings to configure rewards and display options.

== Frequently Asked Questions ==

= Does this work without WooCommerce? =
No. This plugin extends WooCommerce, so WooCommerce must be installed and active.

= Can I use the reward features standalone? =
Yes, the reward progress bar and related cart enhancements can be configured independently.

= Does it support FunnelWheel Cloud integration? =
Yes. Premium features such as advanced CRM, PIM, and Funnel integrations are available via the FunnelWheel cloud platform.
